📊 Student Result Management System

A simple and responsive web-based Student Result Management System built using HTML, CSS, and JavaScript. This project helps manage student records, calculate results, and generate grades dynamically using browser storage.

🚀 Features

    ➕ Add student details (Name & Marks)
    
    📊 Automatic total, percentage calculation
    
    🎓 Grade generation system
    
    🔍 Search / Find student result
    
    🗑️ Delete student record
    
    💾 Data stored using Local Storage
    
    📱 Responsive UI design

🛠️ Tech Stack

    HTML5
    
    CSS3
    
    JavaScript 
    
    Bootstrap
    
    Local Storage API
